# Guide des notifications

Le centre de notifications regroupe les messages locaux produits par Sallon-ConnecT.

## Niveaux

| Niveau | Signification |
|---|---|
| info | Information generale |
| success | Action terminee correctement |
| warning | Situation a verifier |
| error | Erreur a corriger |
| security | Evenement lie a la securite |

## Lire les notifications

Ouvrez la section `Notifications`. Les messages recents apparaissent avec leur niveau et leur date.

## Filtres

Utilisez les filtres pour afficher uniquement certains niveaux ou types de messages.

## Marquer comme lu

Une notification peut etre marquee comme lue depuis son action de ligne. Le compteur de notifications non lues diminue.

## Vider les notifications

Le bouton de nettoyage supprime les notifications locales. Cette action ne supprime pas les logs systeme externes.

## Notifications navigateur

Les notifications navigateur sont optionnelles. Elles demandent une autorisation explicite du navigateur.

## Aucun cloud

Sallon-ConnecT n'utilise pas de notification cloud obligatoire. Les notifications restent locales au projet et au navigateur.

## Securite

Les messages sont nettoyes pour eviter l'affichage de secrets, chemins complets, IP completes ou identifiants sensibles.
