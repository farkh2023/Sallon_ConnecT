# Guide des profils

Les profils adaptent l'interface et les permissions visibles dans Sallon-ConnecT.

## Profils disponibles

| Profil | Usage |
|---|---|
| Principal | Administration locale du hub |
| Famille | Consultation et usages courants |
| Invite | Vue limitee et prudente |
| TV | Affichage grand ecran avec interactions reduites |
| Diagnostic | Verification technique sans actions dangereuses |

## Changer de profil

Utilisez le selecteur de profil dans la barre du haut. Le changement est local et peut modifier les sections visibles.

## Permissions

Chaque profil dispose d'un jeu de permissions. Certaines actions peuvent etre masquees ou refusees selon le profil.

Exemples :

- Un profil TV peut limiter les panneaux sensibles.
- Un profil Invite peut etre en lecture seule.
- Un profil Diagnostic peut voir des informations techniques sans restaurer une sauvegarde.

## Sections visibles

Les sections peuvent varier selon le profil actif. Si une section manque, verifiez le profil courant.

## Mode lecture seule

Le mode lecture seule permet de consulter sans declencher d'action sensible.

## Principe important

Un profil ne contourne jamais les garde-fous backend. Si le backend refuse une action, le profil ne peut pas forcer son execution.
