# Sallon-ConnecT v0.4.0

**Date de release :** 2026-05-22
**Type :** Release locale stable
**Statut :** Stable local-first, packaging final
**Tag propose :** `v0.4.0`
**Node recommande :** `>=22.13.0`

---

## Resume

`v0.4.0` stabilise Sallon-ConnecT comme release locale portable. Cette version consolide les phases 26 a 33 : aide et statut systeme, observabilite et evenements, persistance locale, SSE local securise, centre de notifications, diagnostic avance et packaging release final.

La release reste 100 % locale : aucun cloud obligatoire, aucune telemetrie externe, aucun secret inclus dans les artefacts.

---

## Points forts

- Frontend Next.js valide par 228 tests.
- Backend Express valide par 114 tests.
- Observabilite integree dans le dashboard.
- SSE local via `/api/events/stream`, limite a localhost.
- Notifications locales groupees et filtrees.
- Diagnostic avance via `/api/diagnostics/overview`.
- ZIP portable verifie avec exclusions de secrets, caches, logs, runtime et dependances.
- Scripts release Windows avec lint, tests, build, packaging, checksum et rapport.

## Verification

Commandes obligatoires :

```powershell
pnpm lint
pnpm test
pnpm build
pnpm test:backend
pnpm release:build
```

Le script `verify-release.ps1` inspecte le ZIP et confirme :

- structure attendue ;
- absences d'entrees interdites ;
- diagnostics local-only ;
- SSE limite a localhost ;
- presence de `backup` dans le hook diagnostic ;
- absence de secrets probables ;
- checksum SHA256 et taille ZIP.

---

## Limites connues

- Node.js doit etre installe sur la machine cible.
- Les dependances `npm` ne sont pas embarquees dans le ZIP.
- Le build `.next` n'est pas inclus dans l'archive ; `start-release.ps1` peut le regenerer.
- Le mode multi-machines n'est pas encore implemente.
- SSE reste unidirectionnel ; WebSocket reste une future option.
